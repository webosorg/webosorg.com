## Install

#### Operating system
It was tested only on Linux.

#### Dependencies
 - Nodejs
 - Npm or Yarn
 - Git

#### Clone

```
git clone https://github.com/webosorg/webosorg.com.git
cd webosorg.com
```

#### Global packages
None

#### Install dependencies
```
npm i
```

#### Run
```
node app
```
